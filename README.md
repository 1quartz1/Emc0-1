# Emc0-1
一个正在持续更新的Minecraft模组，我们为游戏重新添加了其他的物品

Emc0=1 Mod 玩家指南
Mod 简介
本 Mod 添加了三个新功能：可以自定义合成配方的魔法棒、可以反向拆解物品的拆解台，以及一颗吃了会触发 雌堕 效果的小药片。

物品/方块列表
✦ 自定义合成表（魔法棒）
用于在游戏中动态创建新合成配方
拥有 100 点耐久，每次创建配方消耗 1 点
右键打开合成配方设定界面
✦ 拆解台（功能方块）
用于将已合成出的物品还原为原材料
硬度适中，需要用镐子挖掘才会掉落自身
如果被拆解的物品带有附魔，会额外返还一本附魔书
✦ 戊酸雌二醇片
可食用的粉色小药片，最多堆叠 64 个
食用后获得 雌堕 效果（2 分钟）
效果持续期间攻击力减半
合成配方
自定义合成表（魔法棒）
材料	摆放位置
铁锭	上、下、左、右（十字形四端）
钻石	正中间
合成台 3×3 网格：中间放钻石，上下左右各放一个铁锭，四个角留空。

拆解台
材料	摆放位置
石头	上、下、左、右（十字形四端）
工作台	正中间
合成台 3×3 网格：中间放工作台，上下左右各放一块石头，四个角留空。

戊酸雌二醇片（一次合成 2 颗）
材料：糖 × 2 + 红蘑菇 × 1
方式：无序合成，放工作台任意位置即可
玩法说明
自定义合成配方
手持魔法棒右键，打开设定界面
在界面中点击背包里的一个物品作为合成目标
从随身物品中左键放入 3×3 合成格（作为材料）
设置产出数量（1-64）
点击 "创建合成表"
新配方会立即同步给服务器上所有在线玩家
之后就可以在工作台按新配方合成了
拆解物品
放置拆解台并右键打开
点击背包中的一个已合成的物品（如工具、方块等）
设定拆解数量
点击 "拆解" 按钮
系统会自动查找这个物品的合成配方，返还对应数量的原材料
如果拆解的是附魔物品，会额外返还一本附魔书
背包满时物品会掉落在地上
雌堕效果
吃下"戊酸雌二醇片"后获得，持续 2 分钟
效果图标为粉色心形
效果持续期间攻击力降低 50%
属于负面效果，可用牛奶桶消除
注意事项
魔法棒设定配方后，所有在线玩家都会收到新配方——请勿用来刷无聊的恶作剧配方
拆解台只能拆解有对应合成配方的物品。原版直接生成的物品（如部分装饰性方块）无法拆解
拆解时必须满足完整的一组才能操作（例如配方产出 4 个，则需要至少 4 个输入物品才能拆解一次）
戊酸雌二醇片是食物，饿不饿都可以吃（alwaysEdible）
所有三种物品都在 "Emc0=1" 创造模式物品栏标签页中
